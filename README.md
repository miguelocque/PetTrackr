# PetTrackr

PetTrackr is a Spring Boot application that helps pet owners manage veterinary visits, medications, feeding schedules, and pet profiles, with an emphasis on clean domain modeling and testable business logic.

## Core Use Cases
- Manage multiple pets per owner
- Track immutable veterinary visit history
- Assign and manage medication schedules
- Maintain feeding schedules and logs
- Generate QR-based lost-pet flyers with public read-only access

## Design Notes
High-level design documents live in `/docs` and are used to guide implementation before coding.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 (local development)
- JUnit / Mockito
